Idea For The Making : I got the Idea to make this Game after remembering way back to 1985, My Parents Bought me an Atari 5500 And It came with the game " Tetris! " I loved that game. In 2025 I thought how cool it would be to Create My Own Tetris Game only Implementing Features the original game Doesn't Have. I Created This Game For Android Devices Only...
About The Author:: My name is Michael Hardy and I'm the Author of The Ultra-Blocks Game and App. I enjoy Walking, Camping, Movies and etc... I have been Programming since 1989 and started creating Software on an Apple IIe in 1989. In 1994 I begin to develop Software and Game's For Windows. Programming is my main hobby and I do it mainly for fun however, I do have Some Published Titles that I sell through Pay-Pal and other Distributors.
Development Tools Used : I Created The Ultra-Blocks Game Completely On An Android Phone Using The AIDE Pro. App. . After I Compiled and Built the App I then Used APK Editor Pro On Android as well as MT Manager Pro for Android To Tweak The App and To Make Some Minor Changes. The Graphics Contained Within The App I Used PicsArt Pro and PhotoRoom Pro Two Graphic App's On Android... For the Launcher Icons I used an Online Android Launcher Icon Generator...
