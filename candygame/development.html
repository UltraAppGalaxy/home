About The Development Of This Windows ® Computer Game ?

This Microsoft Windows ® Computer Game (Candy Volcano) Was Created By Me (Michael  Hardy) Using C++...

The Compiler Used Was MingW32 (GCC) and My Own Integrated Development Environment Called " Creative C++ "

Creative C++ can be obtained for Free from http://www.compiler.ultraappgalaxy.com

Candy Volcano has been in Development Since October of 2005. Since its first release it has became a full blown project of mine.

The Game's Source Code is About 13,000 Lines of Code, It will successfully Compile Using Ceative C++  Or Any MingW GCC Compiler or integrated development environment such as the Code::Blocks IDE or CodeLite IDE. It uses Cross Platform Libraries such as SDL, SDL_IMAGE and FMOD for sounds and music. However, I believe in order to compile this game on a Macintosh you'd probably have to use mikmod or something Similar. Candy Volcano is based off of the Old Mac Os X Game that was produced in 1996 by John Stiles.

All of the tools used to develop this game are Free and can be found on the Internet...
