About The Author Of This Game ?


My name is Michael Hardy and I'm the Developer, Creator and Programmer Of
Candy Volcano. I reside in Missouri...
I have been Programming Since 1989. I first learned Computer Programming on an old Apple //e Computer using AppleSoft Basic in 1989. In 1998 I received my first Windows Computer Running Windows 95. I then started learning the Delphi and C/C++ Programming Languages.

My family and I are NOT Rich but We're Rich in Spirit!
Candy Volcano Was My Very First C++ Project.

On January 22nd, 2008 My Dad (James A. Hardy) past away. This Crushed me but over the past 3 Years or so I've been able to pick up the pieces little by little.

I got the idea to create Candy Volcano in 2005 After Playing a Similar Game On My Friends Macintosh Computer. I liked the game he had so in 2005 I decided to try and create my own copy. I was fairly new to C/C++ so it was a little difficult to make when I first started the project. Most of the Graphics and Sprites were found on the internet.
The others were made by me. I created this game in MingW GCC Using My Very Own Integrated Development Environment Called " Creative C++ " Which is available for Free at http://www.compiler.ultraappgalaxy.com

Well that just about sums it up for who I am, so until next time take care of your self and each other.
